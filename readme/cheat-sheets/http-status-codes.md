# HTTP status codes

Use this page as a quick reference for common HTTP status codes.

Use this when you need to interpret brute-force, crawl, or browser results quickly.

### Common codes

* `200` — exists and is accessible
* `301` — redirect
* `302` — temporary redirect
* `401` — authentication required
* `403` — exists but is forbidden
* `404` — does not exist
* `500` — server error
