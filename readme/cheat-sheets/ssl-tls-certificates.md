# SSL/TLS certificates

Use this page as a quick reference for certificate checks during web enumeration.

Use this when you need names, domains, or contact details from HTTPS.

### Quick check

1. Browse to `https://<TARGET_IP>`.
2. Click the padlock.
3. Open the certificate details.

### What to note

* company name
* email addresses
* domain names
* anything useful and in scope for follow-up
