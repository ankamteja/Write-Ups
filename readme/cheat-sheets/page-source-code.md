# Page source code

Use this page as a quick reference for page source review.

Use this when you need to inspect client-side hints that are not visible in the rendered page.

### Quick check

Press `Ctrl+U` in the browser.

### What to look for

* developer comments
* hidden form fields
* API keys
* internal IP addresses
* file paths
