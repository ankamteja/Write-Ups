# WhatWeb

Use this page as a quick command reference for WhatWeb.

Use this when you need to fingerprint frameworks, versions, and common web technologies.

### Scan one host

```bash
whatweb <TARGET_IP>
```

### Scan a subnet

```bash
whatweb --no-errors <TARGET_SUBNET>/24
```

### What to note

* web server and version
* frameworks and CMS
* email addresses and country data
