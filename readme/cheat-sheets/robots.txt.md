# robots.txt

Use this page as a quick reference for `robots.txt` checks.

Use this when you need to find blocked paths, private directories, or admin routes.

### Quick check

Browse to:

```
http://<TARGET_IP>/robots.txt
```

### What to note

* hidden paths
* admin pages
* private directories
* sensitive files

### Example

```
Disallow: /private
```
