# Staff and Job Postings

* Use staff OSINT to infer the target's tech stack, frameworks, and security tooling.
* Check LinkedIn, company bios, conference talks, and job posts for employee details.
* Review public profiles for clues about cloud providers, internal tools, and deployment habits.
* Search GitHub orgs and personal repos for hardcoded secrets, JWTs, API keys, and exposed emails.
* Correlate names, roles, and code activity to identify useful targets and likely attack paths.
